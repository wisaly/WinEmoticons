WinEmoticons
============

萌颜文字
萌颜文字是一个windows上的外挂式颜文字输入工具。
设计的初衷是可以通过快捷键弹出的窗口中用鼠标点击来选择想要输入的颜文字。

开发工具：VS2008+MFC

项目仍在施工中。
